# mastermind

## Mastermind (Ruby CLI Game)

## Overview
This is a command-line implementation of the classic Mastermind game, built in Ruby using Object-Oriented Programming (OOP).

## Features
✅ Generates a secret code  
✅ Accepts player guesses  
✅ Provides feedback on exact and partial matches  
✅ Validates user input  
✅ Detects when the player wins
